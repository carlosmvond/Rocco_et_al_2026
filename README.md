# Rocco_et_al_2026

This repository contains a single Google Colab notebook,
Rocco_et_al_2026.ipynb, which supports the analyses and figures presented in the manuscript:

**Integrating dynamic modeling and continuous cultivation to enhance recombinant β-glucosidase production in *Komagataella phaffii***

Eduardo Rocco-Martínez<sup>1†</sup>, Johan Quezada<sup>1†</sup>, Julio Berrios<sup>1,3</sup>, Kenneth H. Wolfe<sup>2</sup>, Alvaro Díaz-Barrera<sup>1,3</sup>, Carlos Martínez<sup>1</sup>, Stephanie Braun-Galleani<sup>1,3*</sup>

📘 Contents

The notebook reproduces all computational steps used in the study, including:

Processing of experimental batch data at different temperatures

Dynamic model formulation and parameter estimation

Model–data comparison and goodness-of-fit analysis

Chemostat simulations and productivity analysis

Selection of dilution rates for experimental design

All results can be reproduced by running the notebook sequentially in Google Colab.

🚀 Usage

Open Rocco_et_al_2026.ipynb in Google Colab and run all cells.
All required Python packages are installed automatically.
